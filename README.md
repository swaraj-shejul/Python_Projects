# Number Guessing Game

A multiplayer command-line game built using Python.

## Game Rules

1. Each player selects a secret number between 1 and 20.
2. Numbers are hidden using the getpass module.
3. Players take turns guessing numbers.
4. If a player guesses another player's secret number:
   - That player wins
   - The guessing player loses.

## Concepts Used

- Python Data Types
- Lists
- Dictionaries
- Loops
- Conditional Statements
- getpass module

## How to Run

bash
python number_guessing_game.py
Example

Players: Alice, Bob

Alice secret number: 7
Bob secret number: 14

If Bob guesses 7 → Alice wins.
